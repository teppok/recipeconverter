Recipe converter plugin for Firefox and Safari

Have you ever found a nice recipe on the Internet, but alas, it was in American units. You have to go ask Google or something to convert them to units the rest of the world uses when you should be cooking.

Hope's not lost! The recipe converter is here. This Firefox plugin will automatically convert any American weight or volume units on any page. It's designed for recipe use, so it will understand phrases like "2 to 3 cups of milk" and fractions such as "1 1/2 lbs bacon". However, it will happily convert currency pounds to kgs too, so you may want to keep the add-on on autorun mode only when browsing recipes.

All feedback to teppo dot kankaanpaa at iki.fi is appreciated!

Instructions for use for Firefox:

  • Install the Firefox plugin and restart Firefox.
  • Automatic conversion is not on by default, you can turn it on by clicking the fork icon in the add-on bar at the bottom of the Firefox window (not visible by default - turn it on with View -> Toolbars -> Add-on bar (Näytä -> Työkalupalkit -> Lisäosapalkki in Finnish)) or navigating to Tools -> Add-ons, choosing Extensions side tab, clicking Options on Recipe converter and checking Autorun.
  • Even if automatic conversion is not on, you can convert any page by choosing menu item Tools -> Run recipe converter.

Instructions for use for Safari:

  • Install the Safari plugin. Note that the install is quite silent, so if you get install errors, check that the plugin didn't already get installed.
  • Automatic conversion is not on by default. You can turn it on from Preferences -> Extensions -> Recipe converter -> Autorun.
  • Even if automatic conversion is not on, you can add a button to the toolbar to manually convert any page you are currently viewing. Choose Customize toolbar under the Settings button and drag the Recipe converter fork icon to the toolbar.
  • Safari extension is still in early development and may have performance issues, so you may want to completely disable it when you're not browsing recipes. Uncheck Preferences -> Extensions -> Recipe Converter -> Enable Recipe converter.

  • This program is still in development, so it might be buggy.
  • Known issues:
    • Some patterns are not yet supported. Please let me know if you see anything that isn't converted properly and I might add it.
    • No way to revert back to the original page other than disabling the automation and reloading the page.
    • The program might have performance issues, so keep the automatic conversion disabled for normal non-recipe surfing, or keep an eye out for sluggishness especially if you use a slower computer, and let me know if it happens.
    • Length and temperature conversion will be added at a later date.
